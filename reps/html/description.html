Общее описание

Отличный мультиплатформенный эмулятор для Windows
Достоинства - высокая точность эмуляции, отсутствие game-hack'ов и великолепная совместимость с играми. Для получения звука максимально близкого к оригиналу, используйте опцию: Sound->SuperHQ. Главный же недостаток эмулятора, это продолжение его достоинств - высокие системные требования (для эмулятора Genesis).
